Boolean Data Type in Java

The boolean data type is used to store only two values: true or false.

Example Program

public class BooleanDemo {
    public static void main(String[] args) {

        boolean isJavaEasy = true;
        boolean isRaining = false;

        System.out.println(isJavaEasy);
        System.out.println(isRaining);
    }
}

Important Points

  • ✔ Only true or false are allowed
  • ✔ Used in conditions and decision making
  • ✔ Important Note Unlike C and C++, Java does not treat 1 as true and 0 as false. A boolean variable can store only true or false.
Example:
boolean canVote = age >= 18;

Char Data Type in Java

char is used to store a single character (2 bytes Unicode).

What it can store

  • Uppercase letters (A–Z)
  • Lowercase letters (a–z)
  • Digits (0–9)
  • Symbols (@, $, %, etc.)
  • Unicode characters (Telugu, Hindi, Chinese)

Example Program

public class CharDemo {
    public static void main(String[] args) {

        char english = 'A';
        char telugu = 'అ';
        char hindi = 'क';
        char symbol = '@';

        System.out.println(english);
        System.out.println(telugu);
        System.out.println(hindi);
        System.out.println(symbol);
    }
}

Use Cases

  • Grade system: char grade = 'A';
  • Gender code: char gender = 'M';
A char value must always be enclosed in single quotes (' ')
