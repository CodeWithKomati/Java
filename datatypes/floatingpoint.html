Why Float Uses f or F in Java?

What is Float?

float is a primitive data type used to store decimal numbers.

Data Type Size Precision
float 4 Bytes 6-7 Digits
double 8 Bytes 15-16 Digits

Why Do We Need f or F?

In Java, every decimal number is considered a double by default.

float num = 10.5;

The above code gives a compilation error because 10.5 is treated as a double.

Incorrect Example ❌

float price = 10.5;

Error: Type mismatch. Cannot convert from double to float.

Correct Example ✅

float price = 10.5f;

float pi = 3.14F;

The suffix f or F tells Java that the number is a float literal.

Java Interpretation

Value Java Treats As
3.14 double
3.14f float
100.0 double
100.0F float

Alternative Using Type Casting

float num = (float)10.5;

This works, but using f is easier and more common.

Easy Rule to Remember

Decimal numbers are double by default in Java.

If you want a float value, always add f or F at the end.
3.14 → double
3.14f → float
99.99 → double
99.99F → float
