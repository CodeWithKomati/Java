Java Variables

Learn Variables, Declaration, Initialization, Naming Rules & Examples

1. What is a Variable?

A variable is a name for a memory location used to store data . OR Variables are containers for storing data values . The variable name helps us access the stored value later.

age
18

Click the button to change the value stored inside the variable.

2. Variable Declaration

Declaration means creating a variable by specifying its data type and name. At this stage memory is reserved but no value is assigned.

int age; String name; double salary;

Here, the variables are declared but not initialized.

3. Variable Initialization

Initialization means assigning a value to a variable.

age = 18; name = "Kumar"; salary = 25000.50;

You can also declare and initialize together.

int age = 18; String name = "Kumar";

4. Declaration vs Initialization

Declaration Initialization
Creates a variable. Assigns a value.
Specifies type and name. Specifies actual data.
int age; age = 18;

5. Variable Naming Rules

  • Must begin with a letter, _ or $
  • Cannot begin with a number
  • Can contain letters, digits, _ and $
  • Cannot contain spaces
  • Cannot contain special symbols like @, #, %, -
  • Cannot use Java keywords
  • Java is case-sensitive

6. Naming Conventions

camelCase (Recommended)

studentName totalMarks userAge

snake_case

student_name total_marks user_age

Java commonly uses camelCase for variable names.

7. Length Considerations

Choose meaningful names.

Good

studentName accountBalance emailAddress

Poor

a x abc tmp

8. Valid Variable Names

age studentName userAge _count $price num1 firstName isLoggedIn courseName totalMarks

public class Main { public static void main(String[] args) { int age = 18; String name = "Kumar"; System.out.println(name); System.out.println(age); } }
